Downloading the Client

To use JGET, you will need a client that can consume the API provided. There are two options availible.


Python client

If you are planning on using JGET on your regular desktop (perhaps because you prefer to develop there), then the python client is recomended. There is a windows installer or an executable if you prefer.

If you are on linux (or if you do not trust my nefarious looking executable, or if this is how you prefer to work), you can find the source code on my github. I assume, as the gigachad you no doubt are, that you understand how to run/build the source so it works for you.


Lua Client

However, JGET is intended as a package manager for Computercraft, and so it it would be rather useless without a client written in lua. The lua client is a single file and you can download it directly onto craftOS with the following command:

wget https://jget.trevor.business/get_jget/jget.lua


Now what?

Once you have downloaded and installed your client of choice, you can start downloading packages as you wish. Downloading packages does not require you to log in. If you want to upload however, you will need to sign up for an account and log in with the following command:

jget login

Both clients have extensive built-in help, which can be accessed with the "help" command.
